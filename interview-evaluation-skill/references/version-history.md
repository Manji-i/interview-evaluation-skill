# 版本记录

## v1.1

发布时间：2026-05-25

核心变化：

- 新增维度模版能力：支持把一组面试考察维度保存为可复用模版。
- 新增模版命令：
  - `list-templates`：列出历史维度模版。
  - `show-template`：查看某个维度模版详情。
  - `upsert-template`：新增或更新维度模版。
- 调整工作流：每次面试评价前都必须确认本次维度组合，不能因为历史记忆里有维度就自动套用。
- 支持用户在历史模版基础上增删维度，或新建模版。
- 更新默认 `rubrics.json` schema，新增 `templates` 字段。
- 更新 README、System Prompt、Skill 说明和打包 zip。

适用场景：

- 同一用户针对不同岗位、候选人、职级、面试轮次使用不同考察维度。
- 面试官希望复用常见维度组合，但每次面试前仍保留确认和调整空间。

## v1.0

发布时间：2026-05-21

核心能力：

- 根据飞书妙记、粘贴记录、PDF、Word、DOCX 等材料整理面试评价。
- 支持自定义专业能力和软素质维度。
- 支持 `NH / H- / H / H+ / MH` 评分。
- 支持维度评分标准沉淀和校准。
- 支持外部评分记忆路径 `INTERVIEW_EVALUATION_RUBRICS_PATH`。
- 支持坏 JSON 自动备份恢复。
- 提供可直接上传招聘系统的结构化输出格式。
